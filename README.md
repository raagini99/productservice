Project Name: productservice

This project contains code for the backend of an online edtech + ed-e-commerce application. 

Project specs: 
It uses maven, Java and Spring Boot 3.4.2. 

Dependencies:
Spring Boot Starter Web, Spring Boot Dev Tools, Spring Boot Starter Data JPA, MySQL Connector Java, 
Spring Boot Configuration Processor, Lombok, Spring Boot Starter Test, Spring Boot Starter Actuator, 
Springdoc OpenAPI Starter WebMVC UI. 

Details: 
This application contains code only for backend. It includes controllers and service layer code for 
various endpoints such as POST user, GET user, POST instructor, GET instructor, POST product, 
GET product, etc. and mappings to database layer via models and repositories packages.
